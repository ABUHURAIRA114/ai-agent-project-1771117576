# Console Calculator Project
====================================

## Introduction
---------------

A simple console-based calculator implemented in Python.

## Requirements
------------

* Python 3.8 or higher
* No external dependencies required

## Features
--------

* Basic arithmetic operations: addition, subtraction, multiplication, division
* Supports decimal numbers
* Handles invalid input and divide-by-zero errors

## Usage
-----

1. Clone the repository: `git clone https://github.com/your-username/calculator.git`
2. Navigate to the project directory: `cd calculator`
3. Run the calculator: `python calculator.py`
4. Follow the in-console instructions to perform calculations

## Code Structure
----------------

* `calculator.py`: The main calculator script
* `operations.py`: Defines the arithmetic operations
* `utils.py`: Contains utility functions for input validation and error handling
